[国际象棋pro]

## 简介
- 基于传统国际象棋开发的拓展版国际象棋游戏，棋盘扩展到12*12，新增棋子和玩法。

## 技术栈


嘿 这里有什么新奇的？

- 新的战斗单位！
1.Templar圣殿骑士(T)：以信仰为旗帜的神圣骑士！现在你有一个可以斜着走的骑士了。（B与N的结合）
2.Marshall元帅(M)：王权麾下的最高统帅。不同于其他的领导者，他更愿意冲锋陷阵。（R与N的结合）
3.Assassin刺客(A)：你不会意识到他靠近————直到你意识到不该愣在那里！（2格内任意方向，无视障碍）

- 兵————现在他们更特别了！
1.Spearman长矛兵(SP)：直走直吃，长矛的冲击会粉碎前方的一切！
# 当Spearman冲破地方前线到达对方半场的前两条线时（白方5、6线，黑方即其对称），暂时获得“突围”能力，可以斜向行走（无法吃子）
2.Lineguard阵线护卫(LP)：横移或后撤并非退缩，而是对一切意欲破阵的敌人严阵以待（前后左右各可移动一格，吃子与走子相同）
